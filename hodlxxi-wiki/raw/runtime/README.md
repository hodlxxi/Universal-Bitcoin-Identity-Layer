# Raw Runtime Evidence

This folder stores runtime evidence snapshots used for `runtime-verified` claims.

- Treat files here as append-only evidence artifacts.
- Do not rewrite captured evidence after creation; add a new timestamped file instead.
- Example filenames:
  - `health-2026-04-06.json`
  - `agent-capabilities-2026-04-06.json`
  - `staging-smoke-2026-04-06.txt`

Wiki pages should reference these files when promoting claims to runtime-verified.
