What this proves

When the raw verifier returns valid=true and status=verified, this page can display bounded runtime evidence:

  • the runtime has a signed receipt for the job_id;
  • the receipt signature verifies under the advertised agent key;
  • the receipt records a request hash, result hash, payment hash, timestamp, and related receipt metadata when present.

What this does not prove

This proof is bounded runtime evidence only: not a token sale; not an investment; not KYC; not legal identity; not custody; not a promise of profit; not proof of moral trustworthiness; not a guarantee of future performance; not ownership of a network.
