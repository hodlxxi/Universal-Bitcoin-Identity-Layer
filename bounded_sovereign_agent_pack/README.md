# Bounded Sovereign Agent Pack

## What this pack is

This directory is a repository-grounded implementation dossier for adding **bounded operational sovereignty** to the existing HODLXXI agent surfaces. It documents the current runtime truth, the smallest credible patch path, and the guardrails required before any autonomous execution claims should be made.

## What this pack is not

This pack does **not** implement runtime autonomy, wallet control, root access delegation, or unrestricted execution presented as separate from operator control. It is documentation and verification support only.
